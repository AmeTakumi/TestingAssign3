You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • Example Format: "http://3.142.194.215:9000/guessColorAndNumber?color=green&number=2"
  • /guessColorAndNumber?color=green&number=2
  • Example Format: ""http://3.142.194.215:9000/convertFeetToInches?feet:=12"
  • /convertFeetToInches:=12 ****Please use ONLY a positive whole number (no decimals). Thanks!

File Structure in www (you can use /file/www/FILENAME):

For guessColorAndNumber, your options are numbers 1, 2, or 3, and Colors: red, blue or green.

For convertFeetToInches within "feet:=12" Please do not forget the : ****Please use ONLY a positive whole number (no decimals). Thanks!
